## ğŸ› ï¸ Installation

### Prerequisites
- Web browser (Chrome, Firefox, Safari, or Edge)
- Git (optional, for development)

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Tusharsingh7705/Tic-Tac-Toe_game.git
   cd Tic-Tac-Toe_game
   ```
2. Open `index.html` in your web browser
3. Start playing!

## ğŸ® How to Play
1. Choose your game mode (Player vs Player or Player vs Computer)
2. Select difficulty level if playing against computer
3. Players take turns to place their marks (X or O)
4. First to get 3 in a row (horizontally, vertically, or diagonally) wins!
5. Click "New Game" to restart
